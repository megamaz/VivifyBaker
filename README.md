# Vivify Baker
This is a simple tool that allows you to bake Unity `.anim` file data to Vivify Events.

Roadmap might invole baking more event types in the future.

## Usage
1. Download the `.unitypackage` from the releases tab, and install it.
2. A `VivifyBaker` option should appear at the top of your window.
3. Open the baker you want, and enter the information.
  - The information is currently not saved between uses. Will fix.
4. Hit bake
